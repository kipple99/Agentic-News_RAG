# Agentic News RAG 실행 가이드

## 🚀 빠른 시작

### 1. 패키지 설치

```bash
cd Agentic-News_RAG
pip install -r requirements.txt
```

### 2. Backend 서버 실행

터미널 1에서:

```bash
cd backend
python app.py
# 또는
run_server.bat
```

서버가 `http://localhost:8000`에서 실행됩니다.

### 3. Frontend 실행

새 터미널 2에서:

```bash
cd frontend
streamlit run search_app.py
# 또는
run_app.bat
```

브라우저에서 `http://localhost:8501`이 자동으로 열립니다.

## 🎯 사용 방법

1. **앱 실행 후**
   - 브라우저에서 검색 인터페이스가 열립니다
   - 사이드바에서 Backend 연결 상태 확인

2. **검색하기**
   - 검색창에 질문 입력
   - 예: "오늘 경제 뉴스 검색해줘" (API 검색)
   - 예: "뉴스란 무엇인가요?" (LLM 생성)
   - "Search" 버튼 클릭
   - 에이전트가 자동으로 판단하여 결과 표시

3. **결과 확인**
   - API 검색: 뉴스 기사 카드 형태로 표시
   - LLM 생성: 텍스트 답변 표시

## ⚙️ 설정

### OpenAI API 키

`backend/config.py`에 기본값이 설정되어 있습니다.

또는 환경 변수로 설정:
```bash
set OPENAI_API_KEY=your_api_key
```

### LLM 모델 변경

`backend/app.py`에서 모델 변경:
```python
_agent = create_news_agent(model_type="openai", model_name="gpt-4", use_memory=True)
```

## 🔧 문제 해결

### Backend 연결 오류
- Backend 서버가 실행 중인지 확인
- 포트 8000이 사용 가능한지 확인
- `http://localhost:8000/health` 접속하여 상태 확인

### OpenAI API 오류
- `backend/config.py`에서 API 키 확인
- OpenAI 계정에서 API 사용량 확인
- 네트워크 연결 확인

### 패키지 오류
```bash
pip install --upgrade langchain langchain-openai langchain-community
```

### 포트가 이미 사용 중인 경우
```bash
# Backend 다른 포트로 실행
uvicorn app:app --port 8001

# Frontend 다른 포트로 실행
streamlit run search_app.py --server.port 8502
```

## 📝 참고사항

- **Backend 필수**: Frontend는 Backend 서버가 실행되어야 작동합니다
- **포트**: Backend 기본 포트 8000, Frontend 기본 포트 8501
- **자동 새로고침**: 코드 수정 시 자동으로 반영됨
